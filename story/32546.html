在PC上安装破解Leopard，3步搞定！

2007-10-28 21:48:05

Apple 官方已经正式发布了 Mac OSX Leopard，有黑客已经制作了破解版的 DVD ，可以让我们在任何 PC 上轻而易举地安装 Leopard 系统了！下面就是安装教程，不过，还是要小心为好，硬盘上有数据的话，最好做好备份！

安装之前，你需要做好的准备：

  • 备份重要数据！

  • 破解版的 Leopard DVD 镜像

  • 包含补丁的 zip 文件

  • 一个格式化过的 USB 闪存（磁盘格式为FAT32）

第一步 准备

  • 用刻录软件比如 Nero 刻录 DVD 镜像成单层 DVD 盘片。

  • 格式化 USB 闪存，磁盘标记设置为“Patcher”（不带引号）。

  • 解压缩 Zip 文件放到 USB 闪存里面。

  • 现在你的 USB 闪存里面应该包含了一个名称为“files”的文件夹。

第二步 安装 Leopard

  • 现在你手中已经有了破解版的 DVD，从光盘启动，开始安装！

  • 从光盘启动后，会加载安装向导，需要一些时间，请耐心等待。

  • 选择语言，确信你选择了“Customize（自定义）”，并且对所有显示出来的安装包（Packages）要取消选择。

  • 好了，Leopard 可以安装了，需要一段时间，你可以喝杯咖啡休息一下。

  • 最后会要求你重启，那就重启。重启之前，确信 USB 闪存已经连接到你的PC上了。

第三步 破解 Leopard

  • 现在你已经安装好了 Leopard，该破解了（打补丁）。破解之前，我们需要再次从 Leopard DVD 启动。

  • 等待加载达尔文启动程序（Darwin Bootloader）。一旦加载成功，按F8。你会看到一个弹出框。键入 -s 然后按下回车。DVD 将会以 Verbose 模式装载。注意观察任何错误信息。应该没有什么问题。

  • 装载完成后，选择语言。然后你会看到欢迎屏幕。此时，会转到一个工具终端。

  • 终端加载完毕后，键入如下命令：
    cd ..
    cd ..
    cd Volumes
    cd Patcher
    cd files

  • 现在开始破解了。在终端键入如下命令：
    ./9a581PostPatch.sh

  • 破解程序就会运行。

  • 当破解程序允许完毕，重启就可以了。

第四步  祝贺！你已经完成了“不可能完成的任务了”

注意，破解步骤并没有做过多的额外测试，所以可能你的声卡、网卡不能正常工作。如果出现问题可以通过 OSX86Scene 寻求帮助。注意到，我们并没提供那些破解过的 DVD 镜像和 Zip 文件的下载链接，因为我不想收到 Apple 律师发来的起诉信，你可以通过 Demonoid 来寻找这些文件。

一些更新以及说明：

  • 此破解可运行于 带有 SSE3 指令的 Intel PC

  • 你要将你的 Leopard 分区格式化为 HFS+

  • 确信你安装 Leopard 到 主引导记录（MBR）而不是 GUID
